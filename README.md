
# CP6 - Mobile Application Development

## Integrante
- **Mateus Pierro - RM555125**

## Tecnologias
- React Native (Expo)
- Expo Router
- Moti (animações)
- Expo Barcode Scanner
- SecureStore (armazenamento local)

## Funcionalidades
- Cadastro de produtos
- Leitura de código de barras
- Listagem, edição e exclusão
- Tela de desenvolvedor
- Animações e transições suaves

## Como Rodar
```
npm install
npx expo start
```

## Gerar APK
```
npm install -g eas-cli
eas build -p android --profile preview
```

## Estrutura
- /app → telas
- /assets → ícone e splash
